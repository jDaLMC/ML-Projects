  1. For this data, what are the minimum and maximum number of clusters k one can specify in the cutree function? What will be the corresponding cluster sizes? Explain.
Minimum k: k = 1, cluster size = 64. The minimum k is always 1, because we can't create 0 clusters. k = 1 just means that we are putting all the observations in 1 cluster. So, the size of that cluster would be equal to the number of observations, or 64 in this case.

Maximum k: k = 64, the cluster sizes of each and every cluster is the same and equals 1. The maximum k is n, or the number of observations. In other words, we have n clusters and each of them contain 1 observation. Then, of course, each cluster would have the size of 1.

Question 2

Suppose you have a standardized data matrix \(X\) with \(n\) observations and \(p\) features. What will be the total within-cluster variation (total within-cluster sum of squares) when you implement K-means with \(K=n\)? What will be the corresponding cluster sizes? Explain your answer.

K = n -> there will be n clusters -> there is only 1 observation in each cluster -> there is no variation within each of the clusters -> total within-cluster variation would be 0

The cluster sizes would all be 1, because there is only 1 observation in each cluster.
